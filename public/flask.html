Introduction

Flask is a lightweight Python web framework that helps you build APIs and web apps quickly.

üìò Example: Simple Flask App

from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
    return "Welcome to Dawens Flask Course!"

if __name__ == '__main__':
    app.run(debug=True)

This creates a web server that runs locally on port 5000. Visit http://127.0.0.1:5000/ to see your first Flask page.
