Lesson 1 ‚Äî Variables, Types & Operators

Understanding Python types and how to manipulate them is core. Practice the examples below.

// Types and variables
name = "Dawens"
age = 25
pi = 3.14159
is_active = True

# printing and f-strings
print(f"Hello {name}, age {age}")
Exercise: Create a small script `userinfo.py` that asks user's name and age, then prints a response. 
# solution
name = input("Your name: ")
age = int(input("Your age: "))
print(f"Welcome {name}. In 5 years you'll be {age + 5}.")

Common pitfalls

  • Type conversions: use int(), float(), str().
  • Be careful with integer division (Python 3 uses true division).

Lesson 2 ‚Äî Functions & Modules

Functions help organise code. Keep them small and testable.

def greet(name="World"):
    return f"Hello, {name}!"

if __name__ == "__main__":
    print(greet("Dawens"))
Exercise: Write a function that calculates factorial of a number (iterative). 
def factorial(n):
    result = 1
    for i in range(2, n+1):
        result *= i
    return result

print(factorial(5))  # 120

Tips

  • Use docstrings to document functions.
  • Break large functions into smaller helpers.

Lesson 3 ‚Äî Lists, Tuples, Dicts, Sets

These are the bread and butter of Python data management.

# list
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")

# dict
person = {"name":"Dawens", "age":25}
print(person["name"])

# set
s = set([1,2,2,3])  # {1,2,3}
Exercise: Given a list of numbers, return a dict with counts for each number. 
def counts(lst):
    d = {}
    for x in lst:
        d[x] = d.get(x,0) + 1
    return d

print(counts([1,2,2,3,1,1]))

Lesson 4 ‚Äî OOP basics

Classes, methods, and simple inheritance to structure larger programs.

class Person:
    def __init__(self, name):
        self.name = name
    def greet(self):
        return f"Hello, {self.name}"

class Student(Person):
    def __init__(self, name, grade):
        super().__init__(name)
        self.grade = grade
Exercise: Create a class `BankAccount` with deposit and withdraw methods and balance property.

Lesson 5 ‚Äî HTTP & APIs (requests)

Interact with REST APIs using requests. Parse JSON responses easily.

# simple GET
import requests
resp = requests.get("https://api.github.com")
print(resp.status_code)
print(resp.json().get("current_user_url"))
