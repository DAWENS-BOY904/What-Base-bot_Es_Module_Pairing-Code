Lesson 2: Variables & Data Types

Variables are containers for storing data values. C++ supports several data types like int, float, char, string, and bool.

int age = 21;

Lesson 3: Conditions

C++ uses if, else if, and else statements for decisions.

int age = 18;
if(age >= 18) {
  cout << "You are an adult.";
} else {
  cout << "You are a minor.";
}

Lesson 4: Loops

Loops let you repeat code. The most common are for, while, and do-while.

for(int i = 1; i <= 5; i++) {
  cout << "Hello Dawens " << i << endl;
}
