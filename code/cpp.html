Lesson 1 — Environment & Tooling — الإعداد

Install a C++ compiler (GCC/Clang/MSVC). Recommended: GCC 11+ or Clang. Use an IDE (VSCode, CLion) or online repl (Replit) for quick testing. — ثبّت مترجم C++ واستخدم محرر مناسب.

Tip: On Windows install MSYS2 or use WSL (Ubuntu) for GCC. On Mac use Homebrew: brew install gcc

Lesson 2 — Basics: Hello, Variables, Types — الأساسيات

C++ has primitive types (int, double, char, bool) and user-defined types. Let's start with a minimal program.

// Hello world - C++
#include <iostream>

int main() {
  std::cout << "Hello, JESUS TECH Academy!" << std::endl;
  return 0;
}

Explanation: #include <iostream> gives you input/output streams; std::cout prints to console.

Lesson 3 — Control Flow: if/else, loops — التحكم في التدفق

Master conditions and loops; they form program logic.

    if(i % 2 == 0) std::cout << i << " is even\\n";

Lesson 4 — Functions & Modularity — الدوال

Functions help structure code. Pass by value vs by reference is crucial.

#include <iostream>

// pass by value
int add(int a, int b) {
  return a + b;
}

// pass by reference (modify outside)
void increment(int &x) {
  x++;
}

int main(){
  int x = 5;
  std::cout << add(x,3) << "\\n"; // 8
  increment(x); // x becomes 6
  std::cout << "x=" << x << "\\n";
  return 0;
}

Best practice: keep functions short, single-responsibility.

Lesson 5 — Object Oriented Programming — البرمجة كائنية التوجه

Classes encapsulate state and behavior.

#include <iostream>
#include <string>

class Person {
public:
  std::string name;
  int age;

  Person(const std::string &n, int a) : name(n), age(a) {}

  void greet() const {
    std::cout << "Hello, I'm " << name << ", " << age << " years old\\n";
  }
};

int main(){
  Person p{"Dawens", 25};
  p.greet();
  return 0;
}

Topics: constructors, member functions, access control (public/private), inheritance, polymorphism (virtual functions).

Lesson 7 — STL: Vector, Map, Algorithms — مكتبة القوالب القياسية

STL saves time. Learn containers, iterators, and algorithms.

#include <iostream>
#include <vector>
#include <algorithm>

int main(){
  std::vector<int> v = {4,1,3,2};
  std::sort(v.begin(), v.end());
  for(auto x: v) std::cout << x << " ";
  std::cout << "\\n";
  return 0;
}

Practice: use std::map to count frequencies.

Lesson 8 — File I/O & Serialization — ملفات

Read and write data; remember to check for errors.

#include <iostream>
#include <fstream>

int main(){
  std::ofstream out("data.txt");
  if(!out){ std::cerr<<"Cannot open file\\n"; return 1; }
  out << "Hello file\\n";
  out.close();

  std::ifstream in("data.txt");
  std::string line;
  while(std::getline(in, line)) std::cout << line << "\\n";
  return 0;
}

Lesson 9 — Templates & Generic Programming — القوالب

Templates enable generic data structures and algorithms.

#include <iostream>

template <typename T>
T add(T a, T b){
  return a + b;
}

int main(){
  std::cout << add<int>(2,3) << "\\n";
  std::cout << add<double>(2.5, 1.2) << "\\n";
  return 0;
}

Lesson 10 — Concurrency & Threads — البرمجة المتزامنة

Use <thread> with care. Synchronize with mutexes.

#include <iostream>
#include <thread>
#include <vector>

void worker(int id){
  std::cout << "worker " << id << " start\\n";
  // simulate work
  std::this_thread::sleep_for(std::chrono::milliseconds(200));
  std::cout << "worker " << id << " done\\n";
}

int main(){
  std::vector<std::thread> pool;
  for(int i=0;i<4;i++) pool.emplace_back(worker, i);
  for(auto &t: pool) t.join();
  return 0;
}

Exercise: protect shared data with std::mutex.
